Overview

The POS Stock Management module is designed to enhance inventory control within Odoo’s Point of Sale system by providing real-time stock visibility and threshold-based restrictions. Users can configure stock display to show either on-hand or forecasted quantities, tailoring the system to their operational needs. The module introduces a stock threshold feature that prevents overselling by prompting users with confirmation dialogs when adding products would reduce stock below a defined limit, offering options to adjust quantities or override restrictions. It also supports stock trend indicators to highlight inventory movement, aiding decision-making. Seamlessly integrated with Odoo’s POS interface, the module ensures accurate stock updates, reduces manual errors, and enhances user interaction through dynamic feedback, making it an essential tool for efficient retail and inventory management.


The POS Stock Management module further integrates with Odoo’s inventory system to provide robust stock tracking. When enabled, it automatically updates product quantities in real-time as orders are processed, ensuring consistency between sales and inventory records. This includes detailed logging of product quantities, cart changes, and stock thresholds, offering a clear audit trail for stock movements. By aligning POS transactions with inventory data, the module streamlines operations and supports precise stock control across multiple locations.

Working Flow

  • User have to navigate to Point Of Sale >> Configuration there is a menu Settings.
  • To use the features of the module user has to check the Show Product Stock.
  • There user can configure all the features as per the need.
  • In Stock Display Type there is 2 options.
    On-hand Quantity
    Forecasted Quantity
  • First let's start with the On-hand Quantity.
  • See the on-hand quantity of the product.
  • In POS , for all the products the On-hand Quantity is visible.
  • Now, let's check with the Forecasted Quantity.
  • See the Forecasted quantity of the product.
  • In POS , for all the products the Forecasted Quantity is visible.
  • To show the negative quantity products in POS , Check the Show Negative Quantity Products option.
  • All the products which has Negative or 0 Quantity , it will also appear in the POS.
  • To prevent products with negative quantities from being added to the cart, select the Disable Negative Quantity Products option.
  • All the products with negative or 0 quantity will become Disable.
  • To permit products with negative quantities to be added to the cart, select the Allow Negative Products to Order option.
  • To show the monthly trend of the product, select Show Product Trend.
  • It will show the trend of all the products in last month .
  • if the sale > purchase, the trend will be UP.
  • if the sale < purchase, the trend will be Down.
  • if the sale = purchase, the trend will be Neutral.
  • Also can show the sale and purchase on click of the trend.
  • Set the Threshold Quantity, Which means that minimum that much quantity should remain in the stock.
  • Also can set the custom message, which will display when the quantity leads to less than the threshold.
  • As visible in the below image, that the product has quantity of 16 which is less than the configured threshold quantity 20.
  • When the user try to enter that product into the cart, it will show the popup with custom message as below.
  • If adding a product to the cart results in a quantity below the threshold, a popup will also appear.

Can this module prevent overselling products in the POS?

  • Yes, the module prevents overselling by disabling products with negative or zero stock and alerting users when stock falls below a configured threshold.

Can I choose whether to display on-hand or forecasted stock quantities?

  • Yes, the module allows you to configure whether to display on-hand quantities or forecasted quantities, based on your inventory tracking needs.

Is it possible to customize the stock threshold alerts?

  • Yes, you can set a custom stock threshold and define personalized alert messages to inform users when stock levels are low.

Can I allow selling products with negative stock in certain cases?

  • Yes, the module includes an option to permit ordering products with negative stock, configurable via the POS settings, with user confirmation.

Does the module support stock trend indicators?

  • Yes, the module can display stock trend indicators on product cards, showing inventory movement patterns when enabled in the configuration.
